# 🤝 Contributing to AgentTestPilot

感谢您对 AgentTestPilot 的兴趣！我们欢迎所有形式的贡献。

## 🚀 快速开始

1. Fork 本仓库
2. 克隆您的 Fork: `git clone https://github.com/YOUR_USERNAME/AgentTestPilot.git`
3. 创建分支: `git checkout -b feature/your-feature`
4. 安装开发依赖: `pip install -e ".[dev]"`
5. 进行修改并提交

## 📋 提交规范

我们使用 [Conventional Commits](https://www.conventionalcommits.org/) 规范：

- `feat:` 新功能
- `fix:` 修复问题
- `docs:` 文档更新
- `test:` 测试相关
- `refactor:` 代码重构
- `perf:` 性能优化
- `chore:` 构建/工具变更

## 🧪 测试

- 所有新功能必须包含测试
- 运行测试: `make test`
- 覆盖率要求: >= 80%

## 📝 代码风格

- 使用 Black 格式化: `make format`
- 使用 Flake8 检查: `make lint`
- 行长度限制: 100 字符

## 🐛 报告问题

请使用 GitHub Issues 报告问题，并包含：
- 问题描述
- 复现步骤
- 期望行为
- 实际行为
- 环境信息

## 💡 功能建议

欢迎通过 GitHub Issues 提交功能建议，请描述：
- 使用场景
- 期望的功能
- 可能的实现方案

## 📄 许可证

通过提交贡献，您同意您的代码将在 MIT 许可证下发布。
